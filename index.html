Chapter 5 screening
J1 = P[(M0.04 ≥ 0.8Mp) ∩ (θu ≥ 4%)]
J2 = P[(M0.04 ≥ 0.8Mp) ∩ (θu ≥ 4%) ∩ (σvm,CF/Fy ≤ 1) ∩ (PEEQCF ≈ 0.0)]
J3 = P[(M0.04 ≥ 0.8Mp) ∩ (θu ≥ 4%) ∩ (σvm,CF/Fy ≤ 1) ∩ (PEEQCF ≈ 0.0) ∩ (Mc ≥ 0.8Mc,FS)]
Beneficial targets are treated as lower bounds for strength, energy dissipation, hinge relocation and lower-bound drift capacity. Column-face stress and PEEQ targets are treated as upper bounds in the inverse score. The J-screening logic is separate and uses the updated Chapter 5 definitions above. In the FE-backed dataset, the condition PEEQCF ≈ 0.0 is implemented as PEEQCF = 0.0 within a tiny numerical tolerance. For the drift-capacity term, the interface uses θu when it exists and falls back to θu,LB only for cases where the explicit θu value is censored by the analysed rotation range.
